#Digital Logic LED Controller using SN74HC04N

Instead of simply wiring a button directly to an LED, this circuit uses digital logic and signal:

SN74HC04N is a "NOT" gate. It reads the input signal and outputs the exact opposite. When the button is pressed, it sends a `LOW` (0V) signal to the chip. The chip inverts this to a `HIGH` (5V) signal, which turns on the LED.
