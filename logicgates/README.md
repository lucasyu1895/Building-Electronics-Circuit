Using SN74HC08N(AND), SN74LS86N(XOR), and SN74HC02N(NOR) to build a circuit!!!
In this circuit, we are using these logic gate ic to make the led shine.
Those ICs uses "true table"(0 & 1).

SN74HC08N(AND): 
LED shine when both inputs is 1

SN74LS86N(XOR):
LED shine when inputs are different

SN74HC02N(NOR):
LED shine when both inputs are 0
